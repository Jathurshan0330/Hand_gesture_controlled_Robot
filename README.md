# Hand_gesture_controlled_Robot
Controlling the movements of the Robot using Hand Gestures, Implemented using ATmega32
Hardware Components used are:
            Atmega 32
            Atmega 8
            ADXL 335 , Accelerometer
            L293D Motor Controller 
            HC-06 Bluetooth
            LCD Display
Softwares used:
            Atmel Studio
            AVRDUDESS
            Prothesus
